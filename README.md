# Cybersecurity Compliance Dashboard

## Overview
This project focuses on analyzing cybersecurity compliance data to
identify risks, failed security checks, and overall compliance status.
The dashboard supports audit-ready reporting and security monitoring.

## Key Features
- Compliance status tracking
- Risk indicator visualization
- Trend analysis of security checks
- Audit-ready dashboards

## Tools & Technologies
- Power BI
- SQL
- Excel
- Python (data preparation)

## Project Type
Academic / Learning Project
