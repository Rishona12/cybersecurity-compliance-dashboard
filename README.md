# Cybersecurity Compliance Dashboard
## Overview
This project focuses on analyzing cybersecurity compliance data to
identify risks, failed security checks, and overall compliance status.
The dashboard supports audit-ready reporting and security monitoring.

## Key Features
- Compliance status tracking
- Risk indicator visualization
- Trend analysis of security checks
- Audit-ready dashboards

## Tools & Technologies
- Power BI
- SQL
- Excel
- Python (data preparation)

## Data Handling & Privacy

The original dataset used for this dashboard contains sensitive
information such as IP addresses and security-related identifiers.
To maintain data privacy and ethical standards, the raw dataset
has not been uploaded to this repository.

All analysis and visualizations were performed on sanitized data.
